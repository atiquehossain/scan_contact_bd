# NoNumQR Society Guide

Society admins can create buildings, units, residents, vehicles, parking slots, and visitor logs. Guards can use the mobile app or web guard panel to log visitor entry and exit.

Recommended operational pattern:

- Society admin creates units and resident vehicles.
- Parking QR tags are attached to resident vehicles or slots.
- Guard records visitor name, optional phone, optional vehicle number, destination, entry, and exit.
- Reports show active visitors, vehicles, and residents.
