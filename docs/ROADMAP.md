# Roadmap

## MVP

- OTP auth, profiles, QR tags, public scan pages, contact requests, in-app notifications.
- Sticker shop with Cash on Delivery.
- Admin, reseller, and society panels.
- Flutter QR scanner/generator and core dashboards.
- Docker deployment and backups.

## Production Hardening

- Live SMS gateway.
- bKash/Nagad/SSLCommerz production payment verification.
- External encrypted backups.
- Push notifications with optional FCM/APNs.
- Web application firewall and CAPTCHA for suspicious public traffic.

## Later

- NFC cards.
- Fleet/corporate module.
- Legal call masking integration only after provider approval.
- Advanced abuse/fraud detection.
